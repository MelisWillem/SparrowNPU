# SparrowNPU
This project implements a minimal neural processing unit (NPU) on the PYNQ-Z2 FPGA designed to accelerate quantized transformer models. The NPU provides a custom INT8 matrix-multiplication engine using a small systolic array and on-chip scratchpad memory for high reuse. All heavy linear layers (QKV projections, feed-forward layers) run on the FPGA, while lightweight epilogue operations (residual add, softmax, layernorm, activation) run on the ARM Cortex-A9. The system supports per-tensor symmetric quantization and INT32 accumulation with automatic requantization back to INT8. A reference implementation runs TinyTransformer-style models within the PYNQ Python environment.
